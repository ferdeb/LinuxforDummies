Procesos Para ver los procesos activos en la terminal que estamos ejecutando, usamos: ps Para ver todos los procesos activos en el sistema, usamos: ps -aux Para buscar una palabra clave en los procesos, en este caso root, usamos: ps -aux | grep root Para buscar el process id de un proceso, ejemplo less pidof less Para mostrar una lista de archivos abiertos por procesos en ejecución. lsof Para solo lista procesos que usan la red, usamos: lsof -i Lista los procesos en red, con el puerto 22, o el puerto que queramos especificar este debe ser ejecutado en con permisos de super usuario y nos permite saber que usuarios tienen conexiones encendidas en ssh Si detectamos un usuario desconocido, o que no sea root, podria ser algun intruso en nuestro sistema sudo lsof -i :22
