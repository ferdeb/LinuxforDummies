Post-Install

Post-install se le llama al proceso donde empezamos a configurar nuestra distribucion, empezamos a instalar software, a traves de la terminal Algunos de estos softwares, solo se configuraran una sola vez Estos comandos deben ser ejecutados en Ubuntu Server, y han sido probados hasta Ubuntu Server 24.04

Primero, debemos actualizar los repositorios y el sistema operativo para comprobar que APT este funcionando

Actualizar repositorios: sudo apt-get update

Actualizar el sistema: sudo apt-get upgrade

Activar repositorios multiverse sudo add-apt-repository universe sudo add-apt-repository multiverse sudo add-apt-repository restricted Al finalizar hacemos un apt update y upgrade
