Post-Install

Post-install se le llama al proceso donde empezamos a configurar nuestra distribucion, empezamos a instalar software, a traves de la terminal Algunos de estos softwares, solo se configuraran una sola vez Estos comandos deben ser ejecutados en Ubuntu, y han sido probados hasta Ubuntu 24.04

Este proceso es mediante la terminal, asi que abrimos una terminal Y escribimos el comando necesario para actualizar los repositorios el cual es: sudo apt-get update

Ahora actualizamos los paquetes con el comando: sudo apt-get update
