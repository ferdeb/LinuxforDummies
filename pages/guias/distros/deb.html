Instalación de Debian

Post-Install

Post-install se le llama al proceso donde empezamos a configurar nuestra distribucion, empezamos a instalar software, a traves de la terminal. Algunos de estos softwares, solo se configuraran una sola vez. Estos comandos deben ser ejecutados en Debian, y han sido probados hasta Debian 12. Debian suele tener un post-install algo mas complejo, donde ademas de usar la terminal, deberemos configurar archivos del sistema.

Añadir usuarios sudo (sudoers)

Al instalar Debian, debemos configurar y establecer dos configuraciones, el usuario personal y el superusuario En Debian no esta activado por defecto el modo sudo, es decir los permisos de administrador o super usuario, a traves de nuestro usuario personal Debemos entrar al modo root o super usuario, y realizar unas configuraciones Aunque suene confuso, hacer todas nuestras actividades en directamente en modo super usuario es muy peligroso, si nuestro sistema se ve comprometido, el atacante se encontrara directamente con los permisos totales del sistema Por eso debemos configurar que nuestro usuario personal, pueda acceder a estos permisos mediante la contraseña de superusuario que ya establecimos Y asi siempre nuestro sistema tendra pocos privilegios, lo cual lo hace mas seguro

Activar repositorios non-free

Debian tiene una seria politica con el software que no es libre, para esto debemos activar los repositorios de software "non-free" o software no libre

Actualizar repositorios y actualizar paquetes

Debemos actualizar los repositorios y el sistema operativo para comprobar que APT este funcionando, ademas que cualquiera otro software necesario se instalara teniendo los repositorios non-free activos.

Este proceso es mediante la terminal, asi que abrimos una terminal Y escribimos el comando necesario para actualizar los repositorios el cual es: sudo apt update

Ahora actualizamos los paquetes con el comando: sudo apt update

Thinkpad Laptop Power (Gestor de energia)

TLP (Thinkpad Laptop Power) es una utilidad que nos ayuda a administrar el consumo de energia.

Es una utilidad muy util cuando un dispositivo Linux usa baterias

  1. Instalamos TLP y complementos ejecutando:

    sudo apt install tlp tlp-rdw

  2. Aceptamos la instalacion de TLP:

    sudo tlp start

  3. Sabremos si TLP se instalo correctamente si la terminal regresa a ingresar mas comandos:

    sudo tlp start

  4. Iniciamos TLP con el siguiente comando:

    sudo tlp start

  5. Si obtenemos esta respuesta sabremos que TLP se esta ejecutando correctamente:

    sudo tlp start
