En esta seccion quiero recomendar softwares y utilidades necesarios para el dia a dia Esta lista no sigue un orden especifico, pero cubre mis necesidades, y al conocerlos puede tambien cubrir las tuyas

neofetch

neofetch no es mas que una sencilla herramienta, escrita en Bash, para poder ver las caracteristicas del equipo

Lo instalamos escribiendo en la terminal sudo apt-get install neofetch

Y para usarlo, despues de instalarlo, en la terminal escribes: neofetch En la siguiente imagen podras observar como es ejecutado el comando en Xubuntu y MX Linux.

VLC Media Player El mejor reproductor multimedia de codigo abierto VLC Media Player: sudo apt-get install vlc

Solaar

Solaar es un reemplazo open source para las herramientas de Logitech en Linux Para ofrecer compatabilidad similar a la que tenemos en Windows con su aplicacion nativa LogiOptions+
