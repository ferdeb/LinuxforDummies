Software recomendado para entornos Desktop

En esta seccion quiero recomendar softwares y utilidades necesarios para el dia a dia

Esta lista no sigue un orden especifico, pero cubre mis necesidades, y al conocerlos puede tambien cubrir las tuyas

neofetch

neofetch no es mas que una sencilla herramienta, escrita en Bash, para poder ver las caracteristicas del equipo

Lo instalamos escribiendo en la terminal

sudo apt-get install neofetch

Para usarlo, despues de instalarlo, en la terminal escribimos el comando: neofetch

En la siguiente imagen podras observar como es ejecutado el comando en diferentes distribuciones basadas en Debian.

VLC Media Player

El mejor reproductor multimedia de codigo abierto

Lo instalamos con el comando:

sudo apt install vlc

Solaar

Solaar es un reemplazo open source para las herramientas de Logitech en Linux

Para ofrecer compatabilidad similar a la que tenemos en Windows con su aplicacion nativa LogiOptions+
