Solución a incompatibilidad con paquetes i386 (Arquitectura x86) para instalar Visual Studio Code

Para solucionar el error en cualquier línea de repositorio que nos indique:

  • No es compatible con main/binary-i386/Packages.
  • No admite la arquitectura i386.

Debemos agregar después de la palabra deb en la línea arch=amd64 Ejemplo de línea corregida:

deb [arch=amd64] https://packages.microsoft.com/repos/code stable main
Solucion a los touchpad antiguos (Laptop MX)

Activar TouchPad Synaptic

En la terminal ejecutemos los siguientes comandos:

modprobe -r psmouse modprobe psmouse proto=imps

Para hacer este cambio permanente, creemos un archivo llamado touchpad.conf en la ruta /etc/modprobe.d/ Usando vim o nano, cual sea nuestro editor favorito de terminal sudo vim touchpad.conf /etc/modprobe.d/ sudo nano touchpad.conf /etc/modprobe.d/ y escribimos la siguiente linea de codigo

options psmouse proto=imps

En algunas distros como Xubuntu tendremos el problema de que el servicio de Bluetooth no inicie
Y para solucionar esto tendremos que iniciarlo manualmente desde la Terminal.

Primero, inicia el servicio de Bluetooth systemctl start bluetooth

Segundo, comprueba el servicio de Bluetooth sudo systemctl status bluetooth.service

Opcional, Por si algun motivo deseas detener este servicio systemctl stop bluetooth.
Y recuerda hacer la comprobacion con el comando anterior
